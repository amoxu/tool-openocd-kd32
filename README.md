# The CrossPlatform Binary Pack of the OpenOCD for KD32 MCU

This repository is a binary distribution of the **xPack OpenOCD** for the **KD32** MCU and PlatformIO develop support.

Supported platforms:
- Windows 32/64-bit
- macOS 64-bit/Apple Silicon
- GNU/Linux 64-bit
